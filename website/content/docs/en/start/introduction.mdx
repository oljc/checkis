## Supported Types

checkis provides the following type checking functions:

- `isArray` - Array type checking
- `isBigInt` - BigInt type checking
- `isBoolean` - Boolean type checking
- `isDate` - Date object type checking
- `isEmail` - Email format validation
- `isEmpty` - Empty value checking
- `isError` - Error object type checking
- `isFunction` - Function type checking
- `isInteger` - Integer type checking
- `isNull` - null value checking
- `isNullish` - null or undefined checking
- `isNumber` - Number type checking
- `isObject` - Object type checking
- `isPromise` - Promise object type checking
- `isRegExp` - Regular expression type checking
- `isString` - String type checking
- `isUndefined` - undefined value checking
- `isUrl` - URL format validation
