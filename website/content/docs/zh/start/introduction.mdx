# 简介

**checkis** 是一个现代化的 JavaScript 和 TypeScript 类型检查工具库，提供了全面且高效的类型验证函数集合。

## 特性

- 🚀 **轻量级** - 零依赖，体积小巧
- 🎯 **类型安全** - 完整的 TypeScript 类型定义
- 🔧 **易于使用** - 简洁的 API 设计
- 📦 **模块化** - 支持按需导入
- ⚡ **高性能** - 优化的检查算法
- 🌍 **全面支持** - 覆盖所有常见的 JavaScript 类型

## 支持的类型

checkis 提供了以下类型检查函数：

- `isArray` - 数组类型检查
- `isBigInt` - BigInt 类型检查
- `isBoolean` - 布尔值类型检查
- `isDate` - 日期对象类型检查
- `isEmail` - 邮箱格式验证
- `isEmpty` - 空值检查
- `isError` - 错误对象类型检查
- `isFunction` - 函数类型检查
- `isInteger` - 整数类型检查
- `isNull` - null 值检查
- `isNullish` - null 或 undefined 检查
- `isNumber` - 数字类型检查
- `isObject` - 对象类型检查
- `isPromise` - Promise 对象类型检查
- `isRegExp` - 正则表达式类型检查
- `isString` - 字符串类型检查
- `isUndefined` - undefined 值检查
- `isUrl` - URL 格式验证
